Understanding Class and ID Attributes

Both class and id are HTML attributes used to identify and style any element, but they serve different purposes. class defines a group of elements, while id uniquely identifies a single element.

As a developer, you decide what the assigned values or identifiers should be for classed and ids. These identifiers should be descriptive and follow a consistent naming convention.

The HTML div has a class of box.

<div class="box"> ... </div>

This div has only the class box.

<div class="box"> ... </div>

This is a div with a unique ID "note" and the same "box" class.

<div id="note" class="box"> ... </div>
  • DO NOT use the same ID on multiple elements
  • DO NOT use IDs for styling that applies to many elements
  • DO NOT start class or ID names with numbers
  • DO NOT use spaces in class or ID names (use hyphens or underscores)
